Opdracht 1

Antwoord:

Zoek uit wat transitions zijn en beschrijf met eigen woorden wat het betekent.

CSS Transitions zijn een manier om properties op een element te animeren. Normaal gesproken doet de browser dit voor je en in de meeste gevallen neemt de transitie dan direct plaats.

Met de transition properties heeft de developer controle over hoe de transitie tussen twee states plaats neemt doormiddel van 4 verschillende properties die de animatie bepalen.

transition: <property> <duration> <timing-function> <delay>;

De properties:

transition-property

Specificeert de naam of namen van de CSS properties die geanimeerd moeten worden. Alleen de properties die hier worden toegevoegd worden geanimeerd. Alle ander properties worden direct getransitioneerd zoals gewoonlijk.

transition-duration

Specificeert de duratie of hoe lang een transitie dient plaats te nemen. Je kan een enkele duratie specificeren voor alle properties of verschillende om een transitie interval te creeren.

transition-timing-function

De timing function berekent hoe de waardes van de begin state to de eind state worden berekend. In de praktijk zie je hier het resultaat van in vloeiendheid van de transitie. ease-in bijvoorbeeld begint langzaam en neem dan in snelheid toe. In cijfers zou je dit kunnen weergeven als: 0.1 0.15 0.2 0.6 .1.2.

transition-delay

Definieerd hoe lang een property moet wachten om aan zijn transitie te beginnen.
