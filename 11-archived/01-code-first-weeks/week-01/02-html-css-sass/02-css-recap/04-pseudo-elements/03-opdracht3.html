Opdracht 2

Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

::after

::after kan een element zonder HTML voor een achter element plaatsten

::backdrop

Creëert een box zo groot als de viewport in fullscreen mode.

::backdrop

Creëert een box zo groot als de viewport in fullscreen mode.

::before

::before kan een element zonder HTML voor een ander element plaatsten

::cue

::cue kunnen media captions mee gestyled worden in VTT tracks

::first-letter

Het ::first-letter pseudo element voegt stijlen toe aan de eerste letter van een block element

::first-line

Het ::first-line pseudo element voegt stijlen toe aan de eerste regel van een block element.

::file-selector-button

Het ::file-selector-button pseudo element selecteert de file button van een input met type="file".

::grammar-error (experimenteel)

::grammar-error kan een element zonder HTML voor een ander element plaatsten

::marker (experimenteel)

  • ::marker pseudo element seleteert het markeer boxje van een list item.

::placeholder

::placeholder pseudo element selecteert de placeholder text in een input of textarea

::selection

::selection pseudo element voegt stijlen toe aan door de gebruiker highlighted contents. Zoals muis click & drag over tekst.
