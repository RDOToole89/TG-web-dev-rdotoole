Opdracht 8

Leg in eigen woorden uit wat met cascade en inheritance wordt bedoeld. Maak voorbeelden om je antwoord uit te leggen.

CSS staat voor Cascading Style Sheets. Cascade, specificity en inheritance zijn belangrijke concepten in de wereld CSS.

"The Cascade" is het mechanisme wat er voor zorgt dat een style regel wordt toegepast op een een bepaalt element. Als er twee regels op hetzelfde element van toepassing zijn en deze dezelfde specificity hebben is de regel die als laatste in de stylesheet staat geldig.

Uitleg:

normale selectors dus type / element selectors hebben een specificity van: 0,0,0,1.

.class selectors hebben een specificity van: 0,0,1,0.

#id selectors hebben een specificity van: 0,1,0,0.

style attribute wint altijd met een specificity van: 1,0,0,0.

Voorbeeld:

ul#nav li.active a { property: value; }

specificity: (0, 1, 1, 3)

There are 0 style attributes, there is 1 #id, there is 1 class/ pseudo-class / attribute, there are 3 elements

"Inheritance" is het mechanisme dat reguleert wat voor waarde een een property op een element heeft als er geen specifieke waarde is aangegeven door de developer. Het werkt dus als een soort default waarde. Als een property inheritance heeft wordt dit doorgegeven van parent tot child element. Een voorbeeld hiervan is bijvoorbeeld color. Als dit wordt aangegeven op een parent element zullen de child elementen dit overnemen.

Niet alle properties worden geinherit, het zou bijvoorbeeld niet handig zijn als een property als border inheritance zou hebben.

Het child element inherit in het volgende voorbeeld de kleur rood.

<style> #div1 { color: red; } </style>

<div id="div1"> Parent Div
  <div id="div1Child">Child Div 1</div>
  <div id="div2Child">Child Div 2</div>
</div>

Output:

Parent Div
Child Div 1
Child Div 2
