CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Een CSS regel bestaat uit 3 elementen

h1 { font-size: 1rem; color: orangered; }

Het eerste gedeelte in dit geval de h1 is de selector, daarmee kun je een element, .class of #id mee targeten.

Afhankelijk van de specificiteit wint een selector van een andere selectors.

Het tweede gedeelte is een property in dit geval font-size: , iets wat je wil beinvloeden met css.

Het derde gedeelte is de waarde die je aan deze property will verlenen. In dit geval 1 rem;,
