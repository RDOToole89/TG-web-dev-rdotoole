Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Manier 1: je kunt css direct toepassen op een element door de style attribute te gebruiken. Bijvoorbeeld: <h3 style="width: 200px"></h3>

Manier 2: Je kunt de <style></style> tag gebruiken in de head tag boven aan het document

Manier 3: Je kunt een externe stylesheet gebruiken. Je dient dan wel ernaar toe te linken in je head tag boven aan het document <link rel="stylesheet" href="style.css">
