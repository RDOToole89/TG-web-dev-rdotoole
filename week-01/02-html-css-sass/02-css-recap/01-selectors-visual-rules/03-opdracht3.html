Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld:

  • Type selectors
  • Descendant selectors
  • Class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Type selectors:

Type selectors selecteren elementen aan hun type of "node name". Bijvoorbeeld: a { text-decoration: none; }

Voordeel: Een simpele selector, makkelijk te begrijpen.

Nadeel: Target alle elementen van dat type, dit is niet altijd wenselijk.

Descendant selectors:

Descendant selectors matcht elementen die "descendants" van een bepaald soort parent element zijn". Bijvoorbeeld: div a { text-decoration: none; } of ul.shopping-list li { color: red; }

Voordeel: Je kunt 'nested' elementen selecteren met deze selector.

Nadeel: Target alle elementen die dezelfde 'nesting' structuur vertonen, dit is niet altijd wenselijk.

Class selectors:

Class selector kunnen hergebruikt worden. Daarnaast zijn ze ook vrij specifiek. Bijvoorbeeld: .my-class { font-size: 2rem; }

Voordeel: Specifiek, herbruikbaar. Het is gebruikelijk tegenwoordig om op alle elementen een class toe te passen.

Nadeel: alleen van toepassing op elementen die die .class dragen
