Opdracht 6

Er zijn verschillende selectors, bijvoorbeeld:

  • Child selectors
  • Adjacent selectors
  • General selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Child selector / combinator:

Child selector selector / combinator > selecteren elementen die matchen met de "tweede selector" EN een directe "child" relatie hebben met het eerste element. Bijvoorbeeld: ul > li { margin: 2rem; }

Voordeel: Mogelijk om elementen te selecteren die een parent > child relatie hebben.

Nadeel: Target alle elementen van die aan dezelfde relatie voldoen, dit is niet altijd wenselijk.

Adjacent selector / combinator:

Adjacent selector / combinator + matcht elementen als ze DIRECT volgen na het eerste element en toebehoren aan HETZELFDE parent element. Bijvoorbeeld: h2 + p { color: green; }

Voordeel: Je kunt elementen selecteren die direct naast elkaar liggen.

Nadeel: Target alle elementen die een zelfde relatie hebben. Dit is niet altijd wenselijk.

General selector / combinator:

General selectors ~ matcht elementen als ze volgen na het eerste element in de selector (maar niet per definitie DIRECT volgen) en toebehoren aan HETZELFDE parent element. Bijvoorbeeld: h2 ~ p { color: green; }

Voordeel: Je kunt elementen matchen die elkaar volgen maar geen directe "sibling" relatie hebben.

Nadeel: Target alle elementen die een zefde relatie hebben. Dit is niet altijd wenselijk.
