Maak nu een html bestand aan met voorbeelden van alle soorten selectors erin. Maak een link in je antwoord die naar dat html bestand verwijst.

  • Type selectors selector { property: value; }
  • Descendant combinator selector1 selector2 { property: value; }
  • Class .class-selector { property: value; }
  • Child combinator selector1 > selector2 { property: value; }
  • Adjacent combinator selector1 + selector2 { property: value; }
  • General combinator selector1 ~ selector2 { property: value; }
