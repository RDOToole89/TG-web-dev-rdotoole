Opdracht 1

Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

De default positioning waarden van elementen zijn static. Element volgen normaliter de normale document flow.

Static

Dit is de default waarde voor alle elementen. De top right bottom left en z-index properties hebben geen effect op statische of terwijl default elementen.

Relative

Relative is de positie waar een element relatief staat tot zijn normale positie in de document flow. Wanneer de relative positie wordt gegeven aan een element gebeuren er twee dingen. Je kan gebruik maken van z-index op het element, dit werkt niet bij statische elementen. En child elementen kunnen absoluut gepositioneerd worden in dat blok

Absolute

Absolute positioning betekent dat een element uit de normale document flow wordt verwijderd. Het element wordt dan (by default) relatief in de top left hoek van zijn parent element geplaatst. Zijn positie wordt dan bepaalt door de de top left bottom en left waardes.

Daarnaast wordt er ook een nieuwe stacking context gecreerd als de waarde van de z-index niet op auto staat. Margin collapse treedt ook niet meer op bij absolute gepositioneerde elementen.

Fixed

Een fixed element wordt relatief aan de browser window / viewport gepositioneerd. Het element wordt ook verwijdert uit de normale document flow. Een fixed element blijft op dezelfde positie ook als je scrolled. De positie van die element wordt bepaalt door de waardes van de top right bottom left waardes. Als een parent element transform, perspective of filter toepast zal DAT ELEMENT als containing block functioneren.

box-shadow: rgba(100, 100, 111, 0.2) 0px 7px 29px 0px;

Sticky

Een sticky element wordt relatief aan de browser window / viewport gepositioneerd. Het element wordt ook verwijdert uit de normale document flow. Een fixed element blijft op dezelfde positie ook als je scrolled. De positie van die element wordt bepaalt door de waardes van de top right bottom left waardes.
