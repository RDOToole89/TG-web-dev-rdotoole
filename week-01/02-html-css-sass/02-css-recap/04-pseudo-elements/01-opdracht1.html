Opdracht 1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4) ? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Antwoord:

Met pseudo elementen kan je doomiddel van de :before en :after psuedo classes als het ware twee extra lagen toevoegen aan een element zonder deze in je html markup te hoeven schrijven. Je kan deze elementen absolute positioneren relatief tot hun parent element.

Hierdoor is het mogelijk om bijvoorbeeld meerdere achtergronden toe te voegen aan een element, vormen te creeren, dubbele borders te maken, of een icon voor een stukje tekst of link te plaatsen of overlays te maken.
