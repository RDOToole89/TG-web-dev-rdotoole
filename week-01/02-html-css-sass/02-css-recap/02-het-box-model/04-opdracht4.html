Wat zijn de eigenschappen van block boxes en inline boxes?

Block Elementen

Een block element neemt de volledige breedte in van het "parent" element. Dit element begint altijd op een nieuwe regel.

Inline Elementen

Een inline element begint niet op een nieuwe lijn en neemt alleen zoveel breedte in als nodig is voor het element. Je kan aan een inline element geen hoogte toevoegen.
